# Notices

This local plugin adapts workflow ideas and selected helper scripts from the following repositories:

- JKc66/custom-icons-skill: https://github.com/JKc66/custom-icons-skill
- heroygt/skills, svg-icon-maker: https://github.com/heroygt/skills
- shhac/skills, image-to-svg: https://github.com/shhac/skills

Vendored or adapted material is used to create a PowerPoint-focused Codex workflow:

- monochrome crop, binarization, and trace preparation,
- chroma-key transparent raster cleanup,
- color SVG vectorization from raster sources,
- reference-image decomposition, measurement, and render-comparison discipline.

Review the upstream repositories for their current license files and attribution requirements before redistributing this plugin outside this local environment.
